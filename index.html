ETRI 반도체연구정책센터 · 기술동향 분석

반도체 4대 학회 기술동향 대시보드

국내 KCS 와 해외 3대 학회 IEDM · ISSCC · VLSI 의 발표 논문을 공통 기술분류(19개 영역)에 매핑해 국내·해외 연구동향을 비교합니다.

분석 단위: 논문(세션 영역 상속) · 발행시기 코호트(시즌) = VLSI(Y)+IEDM(Y)+ISSCC(Y+1)+KCS(Y+1)

01

국내 · 해외 비교

기술영역별 비중(%) 차이. 막대가 파란쪽이면 국내(KCS)가, 주황쪽이면 해외가 더 집중.

02

Activity 산점도 (영역 / 키워드)

X = 누적 발표수(연구 풀 크기), Y = 활동지수(최근 비중, 1.0=균등 · >1 성장 · <1 둔화). 4사분면: Emerging(소규모·성장) / Core Rising(대규모·성장) / Niche(소규모·둔화) / Mature(대규모·둔화).

03

시즌별 트렌드

시즌별 기술영역 비중(%) 추이. 범례에서 영역을 켜고 끌 수 있습니다.

04

학회별 연도 트렌드

한 학회 내부의 연도별 기술영역 비중(%) 추이. 학회를 선택하세요. (개최월: VLSI 6월 · IEDM 12월 · ISSCC 2월 · KCS 1~2월)

05

키워드 분석 (국내 vs 해외)

논문 제목에서 반도체 기술 키워드(약 55종)를 매칭. 각 키워드 비중은 그룹별 총 키워드 적중 수 기준으로 정규화(저자줄·OCR 노이즈는 적중 0이라 무영향).

데이터: IEDM·ISSCC·VLSI·KCS 프로그램 PDF에서 논문 추출 후 소속 세션의 공통분류를 상속(규칙 기반). KCS 2025는 OCR 복원. 생성: scripts/build_dashboard_data.py → js/data.js.
