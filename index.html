ETRI 반도체연구정책센터 · 기술동향 분석

반도체 4대 학회 기술동향 대시보드

국내 KCS 와 해외 3대 학회 IEDM · ISSCC · VLSI 의 발표 논문을 공통 기술분류(19개 영역)에 매핑해 국내·해외 연구동향을 비교합니다.

분석 단위: 논문(세션 영역 상속) · 발행시기 코호트(시즌) = VLSI(Y)+IEDM(Y)+ISSCC(Y+1)+KCS(Y+1)

★

핵심 발견 (데이터 자동 요약)

아래 수치는 추출 데이터에서 자동 계산됩니다. 국내 학회(KCS) 무게중심과 해외 학회 내 한국 기관 참여율을 함께 봐야 "진짜 강·약점"이 보입니다.

01

국내 · 해외 비교

기술영역별 비중(%) 차이. 막대가 파란쪽이면 국내(KCS)가, 주황쪽이면 해외가 더 집중.

↑ 막대(또는 02 산점도의 점)를 클릭하면 해당 기술영역의 논문 목록이 여기 표시됩니다.

02

Activity 산점도 (영역 / 키워드)

X = 누적 발표수(연구 풀 크기), Y = 활동지수(최근 비중, 1.0=균등 · >1 성장 · <1 둔화). 4사분면: Emerging(소규모·성장) / Core Rising(대규모·성장) / Niche(소규모·둔화) / Mature(대규모·둔화).

03

시즌별 트렌드

시즌별 기술영역 비중(%) 추이. 범례에서 영역을 켜고 끌 수 있습니다.

04

학회별 연도 트렌드

한 학회 내부의 연도별 기술영역 비중(%) 추이. 학회를 선택하세요. (개최월: VLSI 6월 · IEDM 12월 · ISSCC 2월 · KCS 1~2월)

05

키워드 분석 (국내 vs 해외)

논문 제목에서 반도체 기술 키워드(약 55종)를 매칭. 각 키워드 비중은 그룹별 총 키워드 적중 수 기준으로 정규화(저자줄·OCR 노이즈는 적중 0이라 무영향).

06

기관 분석 — 해외 학회 속 한국 기관

IEDM·ISSCC·VLSI 논문 중 한국 기관(기업·대학·연구소) 참여 논문 비중(저자 소속 기준, 공동연구 포함). "영역별 참여율"이 낮은 곳 = 한국이 국제적으로도 약한 분야.

07

한국 역량 포지셔닝

X = 해외 학회 내 한국 참여율(국제 경쟁력), Y = 국내 학회(KCS) 비중(국내 무게중심). 우상=정착 강점 · 우하=산업 주도(국제 강·국내 학회 약) · 좌상=국내 편중(국제 경쟁력 과제) · 좌하=저활동/공백.

방법론: IEDM·ISSCC·VLSI·KCS 프로그램 PDF → 논문(3,903편) 추출 → 소속 세션의 공통 기술분류(19영역) 상속 → 키워드(약 55종)·한국 기관 매칭. 전체 재생성: python scripts/build_all.py.

한계: ① 규칙 기반 자동 분류(미세 오분류 가능) ② KCS 2025는 이미지 PDF를 OCR 복원 ③ 키워드/기관은 제목·소속 텍스트 매칭이라 미검출(과소계상) 가능 ④ 국내=KCS(국내 학회), 해외=국제 학회라 발표 채널 차이가 섞임 → "기관 분석"으로 보정. 정량 추세 해석용.
