반도체 4대 학회 기술동향 대시보드

국내 KCS 와 해외 3대 학회 IEDM · ISSCC · VLSI 의 세션을 공통 기술분류(19개 영역)에 매핑해 국내·해외 연구동향을 비교합니다.

분석 단위: 세션 · 발행시기 코호트(시즌) = VLSI(Y)+IEDM(Y)+ISSCC(Y+1)+KCS(Y+1)

01

국내 · 해외 비교

기술영역별 비중(%) 차이. 막대가 파란쪽이면 국내(KCS)가, 주황쪽이면 해외가 더 집중.

02

기술영역 Activity 산점도

X = 누적 세션수(연구 풀 크기), Y = 활동지수(최근 비중, 1.0=균등 · >1 성장 · <1 둔화). 4사분면: Emerging(소규모·성장) / Core Rising(대규모·성장) / Niche(소규모·둔화) / Mature(대규모·둔화).

03

시즌별 트렌드

시즌별 기술영역 비중(%) 추이. 범례에서 영역을 켜고 끌 수 있습니다.
