# 반도체 4대 학회 기술동향 — 요약 인사이트

> IEDM·ISSCC·VLSI·KCS 프로그램에서 추출한 **논문 3,903편**(세션 622개)을 공통
> 기술분류(19영역)에 매핑하고, 키워드(~55종)·기관(저자 소속)을 분석한 결과.
> 대시보드: https://juwonetri-eng.github.io/kcs-iedm-vlsi-isscc/

## 1. 학회 성격
- **IEDM = 소자/공정**, **ISSCC = 회로/시스템**, **VLSI = 가교**, **KCS = 소재·공정·설계 종합(국내)**
- 시즌(발행 코호트): `VLSI(Y)+IEDM(Y)+ISSCC(Y+1)+KCS(Y+1)` — 개최월 차이 보정

## 2. 한국의 국제 위상 (해외 3대 학회 기관 분석)
- 해외 3대 학회 논문의 **21.0%**에 한국 기관 참여 (IEDM 19.8% / ISSCC 18.4% / VLSI 25.4%)
- 주도 기관: **삼성 247편**, KAIST 80, SK하이닉스 65, POSTECH 58, 서울대 34 …
  (기업 304 · 대학 274 · 연구소 8) — **삼성이 거의 전 영역 1위**, KAIST가 모델링·바이오메디컬 주도

## 3. 강점 (국내·국제 모두 강함)
| 영역 | 해외 학회 내 한국 참여율 |
|---|---|
| 메모리(DRAM·NAND·차세대 NVM) | **40%** |
| 광전자·디스플레이·이미지센서 | **32%** |
| 전력관리 회로 | 25% |
| 공정·소재·계측 | 24% |
- 키워드로도 ALD/CVD·산화물반도체·강유전·뉴로모픽(소자)·EUV에서 국내 압도적

## 4. 약점 후보 (국제 참여율도 평균 21% 미만 → 실질 격차 가능성)
| 영역 | 해외 참여율 |
|---|---|
| 센서·MEMS·바이오소자 | **10%** |
| 소자 모델링·신뢰성 | 11% |
| **양자·극저온** | 11% |
| 파워·RF 소자 | 12% |
| RF·mmWave·무선 회로 | 13% |
| 보안/HW신뢰 · 와이어라인 I/O | 15% |
- 키워드: RF transceiver, ADC/DAC, SerDes, PLL, FinFET, MRAM에서 국제 비중 낮음

## 5. 해석상 주의 (중요)
- 국내=**KCS(국내 학회)**, 해외=**국제 학회**라 발표 채널 차이가 섞임.
  예) 전력관리는 KCS엔 0편이나 국제 참여율 25% → "국내 학회에 안 낼 뿐 약점 아님".
- 따라서 **"국내 무게중심(01·05)"과 "국제 참여율(06)"을 함께** 봐야 진짜 강·약점이 보임.
  종합 판단 도구가 **07 한국 역량 포지셔닝**(국제 참여율 × 국내 비중) 사분면.

## 6. 시사점 (초안)
- **유지·강화**: 메모리·디스플레이·소재/공정 — 국제 경쟁우위 확고
- **점검 필요**: 양자/극저온, RF·아날로그, 소자 신뢰성/모델링 — 국제 참여율 낮음.
  소자·재료(상류) 강점이 회로·시스템(하류) 구현으로 이어지는지(예: CIM 소자→CIM 회로) 점검.
- 단, 위 약점은 *국제 학회 발표 비중* 기준이며 산업 실적·특허와 별개로 추가 검증 권장.
